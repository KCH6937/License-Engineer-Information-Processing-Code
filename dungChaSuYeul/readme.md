# 등차수열
등차수열은 초깃값과 공차로 이루어져있다.
- - -
초깃값 : a  
공차 : d     
n : n번째 항을 나타냄  
공식 : a + (n-1)d
- - -

ex) 초깃값이 3이고 공차가 5라고 가정하였을 때, 5번째 항의 값은   
3 + (5 - 1) x 5 = 23 인것이다.

코드 예시

```
int a = 3;	// 초깃값
int d = 5;	// 공차
int ret[5] = { 0, }; // 5번째 항을 구하므로 편의상 배열크기를 5로 잡았다.

ret[0] = a; // 1번째 항은 3이므로 a를 대입

for(int i = 1; i < 5; i++)
{
	ret[i] = ret[i-1] + d;	// i번째 항은 i-1번째 항에 공차(d)를 더하면 됨
}

printf("%d", ret[4]);

```
